## Variable in Python

### 1. Integer
Integer is an number that represents from `-Infinity to +Infinity`

<i>Example: </i>
- [x] positiveNumber = 10
- [x] negativeNumber = -9
- [x] age = 30
- [x] numberOfStudent = 25
- [x] numberOfCar = 100
#### Integer conversion
<i>Example: </i>
- [x] string = "10" :arrow_right: number = int(string)
- [x] string = "A" :arrow_right: `ValueError: invalid literal for int() with base 10: 'A'`
